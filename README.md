Proyek Sistem Informasi Geografis (SIG)
​Repositori ini berisi tugas Sistem Informasi Geografis yang dibuat sebagai bagian dari pembelajaran pemrograman web dan pemetaan menggunakan Python.
​Struktur File Proyek
​Di dalam folder ini, terdapat file-file berikut yang merupakan bagian dari tugas saya:
​latihan.ipynb: File utama yang berisi kode pemrograman dan logika pemetaan menggunakan Python.
​dataa.xlsx: File data pendukung (Excel) yang digunakan dalam proyek.
​map.html: File hasil visualisasi peta interaktif yang telah berhasil dibuat.
​README.md: File dokumentasi ini yang menjelaskan isi proyek.
​Penjelasan Singkat
​Proyek ini menggunakan pustaka folium untuk menampilkan peta interaktif berdasarkan koordinat yang ditentukan di dalam latihan.ipynb.
